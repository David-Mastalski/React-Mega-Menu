This project showcases a modern Mega Menu for React, modeled after popular e-commerce platforms. It enables quick and easy navigation while dynamically swapping visual elements to match the currently selected category.


## Installation 
Clone this repository to your local computer
```
git clone https://github.com/David-Mastalski/React-Mega-Menu.git

```
Navigate to the application directory

Open the directory with a code editor such as Visual Studio Code
```
npm i
```
```
npm run dev
```
